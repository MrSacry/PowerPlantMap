# PowerPlantMap

This repository contains the PowerPlantMap project, which allows users to access an interactive map of power plants.

## Accessing the Map
You can view the map directly at the following link:  
[PowerPlantMap](https://github.com/MrSacry/PowerPlantMap/blob/main/assignment_template.html)

## Instructions for Using Locally
1. Clone the repository to your local machine:
   ```bash
   git clone https://github.com/MrSacry/PowerPlantMap.git
   ```
2. Navigate to the repository directory:
   ```bash
   cd PowerPlantMap
   ```
3. Open the `assignment_template.html` file in your web browser to view the map locally.

## Instructions for Using via GitHub Pages
1. Ensure you are in the repository settings.
2. Scroll down to the "GitHub Pages" section.
3. Select the `main` branch as the source and save.
4. After a few minutes, your site will be available at `https://<your-username>.github.io/<repository-name>/`.
   For example: `https://MrSacry.github.io/PowerPlantMap/`

Enjoy exploring the PowerPlantMap!
